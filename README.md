# Квіти Львова - Сайт доставки квітів

Сучасний адаптивний веб-сайт для доставки квітів у м. Львів, створений на Next.js 16 з React 19.

## Основні функції

- **Головна сторінка** з привабливим hero-розділом та переліком переваг
- **Каталог** з 20 квітковими композиціями, пошуком та фільтрацією за категоріями
- **Модальні вікна** для перегляду деталей товарів
- **Кошик** з можливістю зміни кількості, видалення товарів та управління послугою Crystal
- **Автоматичне додавання послуги Crystal** (50 грн) - можна прибрати
- **Авторизація** через localStorage (локальна реєстрація/вхід)
- **Оформлення замовлення** доступне лише після авторизації
- **Логіка доставки**:
  - Сума > 1500 грн → безкоштовна доставка
  - Сума < 1000 грн → доставка 150 грн
  - 1000-1500 грн → доставка 100 грн
  - Самовивіз → безкоштовно
- **Способи оплати**: карткою онлайн або готівкою при отриманні
- **API для збереження замовлень** у JSON файли (без бази даних)
- **Сторінка успішного замовлення** з деталями

## Технології

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Zustand (управління станом)
- Shadcn/ui компоненти
- Sonner (toast повідомлення)

## Встановлення та запуск

```bash
# Встановити залежності
npm install

# Запустити в режимі розробки
npm run dev

# Відкрити у браузері
http://localhost:3000
```

## Структура проекту

```
├── app/
│   ├── page.tsx              # Головна сторінка
│   ├── catalog/              # Каталог товарів
│   ├── cart/                 # Кошик
│   ├── auth/                 # Авторизація
│   ├── checkout/             # Оформлення замовлення
│   ├── success/              # Успішне замовлення
│   └── api/order/            # API для збереження замовлень
├── components/
│   ├── header.tsx            # Шапка сайту
│   ├── footer.tsx            # Футер
│   ├── product-card.tsx      # Картка товару
│   ├── product-modal.tsx     # Модальне вікно товару
│   └── cart-item.tsx         # Елемент кошика
├── lib/
│   ├── products.ts           # Дані товарів
│   ├── cart-store.ts         # Стан кошика
│   └── auth-store.ts         # Стан авторизації
└── orders/                   # Збережені замовлення (створюється автоматично)
```

## Особливості

- Повністю адаптивний дизайн для всіх пристроїв
- Українська локалізація
- Валюта UAH (грн)
- Валідація форм
- Toast повідомлення про дії користувача
- Натуральна зелена колірна схема
- Професійні фотографії квітів
- Автоматичний розрахунок вартості доставки

## Замовлення

Всі замовлення зберігаються у папку `/orders`:
- Окремі JSON файли для кожного замовлення
- Текстовий лог файл з історією всіх замовлень
